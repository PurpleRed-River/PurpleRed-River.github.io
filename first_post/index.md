# 2022年的开始


《我们生活在南京》走到了尾声，压抑中带有一丝平淡，人生也就是这样，终得一死，连翘还需要背负着”老师“之名等待半夏的到来，那枚摩斯码练习币将穿越时间和苦痛的长河见证整篇故事。 我们怀念一切，希望不只是因为”刀客“的到来。而是人性中美好的部分如同星光点缀的夜空，不是因为光亮，是因为漆黑的夜空而让人记忆深刻。
